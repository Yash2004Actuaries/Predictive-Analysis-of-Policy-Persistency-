# Policy Lapse Prediction using GLM.

This project uses Actuarial Statistics (CS1) to predict insurance policy lapses based on age and premium frequency.

# Key Features
* **Data Cleaning:** Used `Tidyverse` for handling missing values and outlier detection.
* **Exploratory Data Analysis (EDA):** Visualized trends using `ggplot2`.
* **Modeling:** Built a **Logistic Regression (GLM)** model to calculate lapse probabilities.

# Technical Summary
* **Link Function:** Logit
* **Variables:** Age, Premium Frequency, Sum Assured.
* **Goal:** To improve the company's **Persistency Ratio**
